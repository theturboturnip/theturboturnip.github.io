Traits

CompressedCapability

Trait defining an Rust version of the public API for a specific capability type. A type X implementing CompressedCapability is equivalent to the API provided by cheri_compressed_cap_X.h in C, where ccx_cap_t is equivalent to CcxCap.

FfiNumType

Value which can be converted to T (a NumType). Must also be Default/Copy/Clone/Debug, so CcxCap can derive these.

NumType

Trait that the field types defined in CompressedCapability (Length, Offset, Addr) have to implement. This asserts that a) they’re numeric, b) they support Default/Copy/Clone/Debug so that CcxCap can derive these.
